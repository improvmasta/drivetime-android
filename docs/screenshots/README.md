# Screenshots

The main `README.md` has commented-out `<img>` blocks reserved for these. Drop a file in
here, uncomment the matching block, and it renders — no other edit needed.

Shoot on a real phone, in **dark theme** (it's the app's better look and what most people
will see at night), with a drive history that looks lived-in rather than empty. Portrait,
roughly 1080×2400. Keep any real addresses out of frame, or rename the places first.

| File | What it shows | Why it's in the README |
|---|---|---|
| `hero.png` | The Drives tab, mid-scroll, ideally with the live HUD visible | The one shot above the fold — it has to say "this is a complete log I didn't have to maintain" |
| `mileage.png` | The Mileage ledger: the reimbursable-$ hero, tagged rows, a suggestion chip | The money shot, literally — this is the feature that sells the app |
| `drive-detail.png` | A drive's detail page: the speed-heatmapped route + the speed chart | Proves the depth behind each row |
| `commute.png` | A route comparison: the fork table with a ★ on the fastest way | The thing no other tracker does |
| `setup.png` | The first-run wizard — the welcome hero or the permissions checklist | Shows how little setup there is |

Nice-to-haves if you're taking more anyway: the notification centre with a couple of pending
items, and the ongoing drive notification on the lock screen.
